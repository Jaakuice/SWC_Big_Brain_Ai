## How to Write Good Questions

The better your question, the better the answer! Here are some tips:

### ‚úÖ DO:

- **Be specific:** "What did Andrew Webley say about Bitcoin in November 2025?" (better than "What did he say?")
- **Include dates when relevant:** "What was SWC's share price on IPO day?"
- **Ask for sources:** "What are the key risks mentioned in the prospectus? Please cite the source."
- **Use clear language:** Write as if you're asking a friend

### ‚ùå DON'T:

- **Don't be too vague:** "Tell me about SWC" (too broad - what specifically?)
- **Don't use jargon you don't understand:** The AI will understand plain English!
- **Don't ask for predictions without context:** Instead of "Will the price go up?", ask "Based on historical trends, what factors have influenced SWC's share price?"

### Examples of Good Questions:

‚úÖ "What is the current sats per share and how has it changed since IPO?"

‚úÖ "Summarize Andrew Webley's view on Bitcoin treasury risks from his latest YouTube appearances."

‚úÖ "What are the key risk factors mentioned in SWC's latest prospectus?"

‚úÖ "How many Bitcoin purchases has SWC made since going public?"

‚úÖ "What did the CEO tweet about on January 15, 2026?"

## Troubleshooting

### Problem: "I can't find the download button on GitHub"

**Solution:**
- Make sure you're on: https://github.com/Jaakuice/SWC_Big_Brain_Ai
- The green "Code" button should be near the top right
- If you still can't find it, try a different web browser

### Problem: "The AI says it can't find the files"

**Solution:**
- Make sure you've opened the correct folder in your AI assistant
- In Cursor: Check that you see folders like "RNS", "Youtube", "Tweets" in the left sidebar
- Try closing and reopening the folder
- Make sure you extracted the ZIP file (don't try to use the ZIP file directly)

### Problem: "The AI gives me wrong information"

**Solution:**
- Ask the AI to cite its sources: "Can you show me which file you got that from?"
- Try rephrasing your question to be more specific
- Check if the information might be in a different file - try asking: "Where can I find information about [topic]?"

### Problem: "Cursor/Claude won't install"

**Solution:**
- Make sure your computer meets the system requirements
- Try downloading again from the official website
- Check if your antivirus is blocking the download
- Try a different AI assistant (they all work similarly)

### Problem: "I don't understand the answer"

**Solution:**
- Ask for clarification: "Can you explain that in simpler terms?"
- Ask for examples: "Can you give me an example?"
- Break down your question: Instead of one big question, ask several smaller ones

### Problem: "The chat isn't working"

**Solution:**
- Make sure you're connected to the internet
- Try refreshing the page (if using a web-based tool)
- Try restarting the application
- Check if you need to sign in or create an account
